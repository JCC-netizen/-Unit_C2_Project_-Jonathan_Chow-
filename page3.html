Lesson 2: Introduction to Basic HTML Tags

In HTML, tags are used to define elements. Some of the most common tags are <p> for paragraphs, <h1> to <h6> for headings, and <table> to create tables. Understanding these tags is essential to structuring your content effectively. Here are a list of HTML tags that web developers use to create their webpages.

Examples of HTML Code:

        <h1>The main element that defines an HTML document</h1>
        <p>defines a paragraph</p>
        <title>Specifies the document's title that will appear in the tab or title bar of the browser</a>
        <body>Includes the text, pictures, and links that make up the document's content</body>
        <ol>Creates an ordered list</ol>
        <ul>Creates an unordered list</ul>
        <a href>Defines a hyperlink</a>
        <img src>Creates an image<>
        <li>Defines a list item within an ordered or unordered list</li>
        <table>creates a table</table>

When developing your webpage, theses types of tag serve their own purpose and will help you develop your webpage. In the lessons ahead, you will learn how to effectively use them and more!
