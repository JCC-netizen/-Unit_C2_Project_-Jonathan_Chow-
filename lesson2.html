Lesson 2: Basic HTML Tags

In HTML, tags are used to define elements. Some of the most common tags are <p> for paragraphs, <h1> to <h6> for headings, and <a> for hyperlinks. Understanding these tags is essential to structuring your content effectively.

Example HTML Code:

        <h1>This is a Heading</h1>
        <p>This is a paragraph</p>
        <a href="https://www.example.com">Visit Example</a>

On the webpage, the <h1> tag will display a large heading, the <p> tag will display a paragraph, and the <a> tag will create a clickable link that redirects the user to "https://www.example.com".
