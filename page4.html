Lesson 3: Structuring an HTML Document

In HTML, most elements are defined using tags, which typically come in pairs: a start tag and an end tag. The start tag begins the element, enclosed in angle brackets and often includes the element's name <p> for a paragraph. The end tag signifies the closure of the element, and it is similar to the start tag but includes a forward slash before the element's name </p>. Together, these tags define the content or structure they enclose. For example:

<p>This is a paragraph.</p>

This line of code will look like this on a webpage:

This is a paragraph.

Every HTML document follows a basic structure. It starts with the <!DOCTYPE html> declaration, followed by the <html> element. Inside <html>, you'll have the <head>, <title>, and <body> sections. In the <body> section, you can have tags like <h1> and <p>. The <head> contains metadata, while the <body> contains the visible content. The <title> specifies the webpage's title, <h1> defines a heading, and <p> defines the paragraph.

Example HTML Code:

        <!DOCTYPE html>
        <html>
            <head>
                <title>My Web Page</title>
            </head>
            <body>
                <h1>Welcome to My Web Page!</h1>
                <p>This is the content of my webpage.</p>
            </body>
        </html>

This structure ensures that the browser knows how to process the document. The content inside <body> will be visible to users on the webpage, while <head> contains information like the page title that appears on the browser tab.

This is how it looks like on a webpage:

Welcome to My Web Page!

This is the content of my webpage.
