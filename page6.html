Lesson 5: Images and Tables

The <img> tag will display an image with the source file "image.jpg," while the <table> tag creates a table. In the <img> tag, there are two parts, "src" and "alt". "Src" is the url or file path of the image while "alt" is a description of the image. In the <table> tag, there are the <tr>, <td>, and <th> tags. The <tr> tag defines a row, the <td> tag defines the data inside the table, and the <th> tag creates headers for the table.

Example of <img> and <table> tags:

    <img src="image.jpg" alt="A beautiful image">

    
This is how it looks like on a webpage!
