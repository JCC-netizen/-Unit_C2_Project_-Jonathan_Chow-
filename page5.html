Lesson 4: Common HTML Elements

There are several HTML elements you will use often in web development. These include images <img>, lists <ul>, <ol>, and tables <table>. These elements allow you to add content that enhances the functionality and presentation of your web pages.

On the webpage, the <ul> tag will create an unordered list, and each <li> tag will create a list item.

Example of <ul> and <ol> tags:

        <ul>
            <li>First item</li>
            <li>Second item</li>
        </ul>

        <ol>
            <li>First item</li>
            <li>Second item</li>
        </ol>

This is how it would look on a webpage:

  • First item
  • Second item
  1. First item
  2. Second item

The <img> tag will display an image with the source file "image.jpg," while the <table> tag creates a table. The <img> tag uses two main attributes: src, which specifies the URL or file path of the image, and alt, which provides a description of the image.

In the <table> tag, there are additional tags like <tr>, <td>, and <th>. The <tr> tag defines a row, <td> defines the data inside the table, and <th> creates headers for the table.

Example of <img> and <table> tags:

        <img src="image.jpg" alt="A beautiful image">

        <table border="1">
            <tr>
                <th>Name</th>
