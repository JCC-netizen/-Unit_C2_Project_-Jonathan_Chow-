Lesson 4: Common HTML Elements

Did you know you can make lists on html? To make lists on html, you will need to know the <li> tag, <ul>,tag and <ol> tag. The <ul> tag will create an unordered list, the <ol> tag will create ordered lists, and each <li> tag will create a list item. These elements allow you to add content that enhances the functionality and presentation of your web pages.

Example of <ul>, <ol>, and <li> tags:

        <ul>
            <li>First item</li>
            <li>Second item</li>
        </ul>

        <ol>
            <li>First item</li>
            <li>Second item</li>
        </ol>

This is how it would look on a webpage:

  • First item
  • Second item
  1. First item
  2. Second item
