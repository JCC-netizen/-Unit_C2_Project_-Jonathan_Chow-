Lesson 4: Common HTML Elements

There are several HTML elements youâ€™ll use often in web development. These include images (<img>), lists (<ul>, <ol>), and tables (<table>). These elements allow you to add content that enhances the functionality and presentation of your web pages.

On the webpage, the <ul> tag will create an unordered list, and each <li> tag will create a list item. The <img> tag will display an image with the source file "image.jpg".

Example HTML Code:

        <ul>
            <li>First item</li>
            <li>Second item</li>
        </ul>

        <img src="image.jpg" alt="A beautiful image">
