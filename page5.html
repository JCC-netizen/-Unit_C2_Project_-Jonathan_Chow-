Lesson 4: Common HTML Elements

There are several HTML elements you will use often in web development. These include images <img>, lists <ul>, <ol>,<img src> and tables <table>. These elements allow you to add content that enhances the functionality and presentation of your web pages.

On the webpage, the <ul> tag will create an unordered list, and each <li> tag will create a list item.

Example of <ul> and <ol> tag:

        <ul>
            <li>First item</li>
            <li>Second item</li>
        </ul>
       
        <ol>
            <li>First item</li>
            <li>Second item</li>
         <ol>

This is how it would looks like on a webpage
