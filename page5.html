Lesson 4: Common HTML Elements

There are several HTML elements you will use often in web development. These include images <img>, lists <ul>, <ol>, and tables <table>. These elements allow you to add content that enhances the functionality and presentation of your web pages.

On the webpage, the <ul> tag will create an unordered list, and each <li> tag will create a list item.

Example of <ul> and <ol> tags:

        <ul>
            <li>First item</li>
            <li>Second item</li>
        </ul>

        <ol>
            <li>First item</li>
            <li>Second item</li>
        </ol>

This is how it would look on a webpage:

  • First item
  • Second item
  1. First item
  2. Second item

The <img> tag will display an image with the source file "image.jpg," while the <table> tag creates a table. In the <img> tag, there are two parts, "src" and "alt". "Src" is the url or file path of the image while "alt" is a description of the image. In the <table> tag, there are the <tr>, <td>, and <th> tags. The <tr> tag defines a row, the <td> tag defines the data inside the table, and the <th> tag creates headers for the table.

Example of <img> and <table> tags:

        <img src="image.jpg" alt="A beautiful image">

        <table border="1">
            <tr>
                <th>Name</th>
                <th>Age</th>
                <th>City</th>
            </tr>
            <tr>
                <td>John</td>
                <td>25</td>
                <td>New York</td>
            </tr>
            <tr>
                <td>Jane</td>
                <td>30</td>
                <td>Los Angeles</td>
            </tr>
        </table>

This is how it looks like on a webpage!

Name Age City
John 25 New York
Jane 30 Los Angeles
