Lesson 4: Creating Lists

There are two HTML elements you will use often in web development to create lists. These include images <ul> and <ol>. These elements allow you to add content that enhances the functionality and presentation of your web pages.

On the webpage, the <ul> tag will create an unordered list, the <ol> tag will create an ordered list, and each <li> tag will create a list item.

Example of <ul>, <ol>, and <li> tags:

        <ul>
            <li>First item</li>
            <li>Second item</li>
        </ul>

        <ol>
            <li>First item</li>
            <li>Second item</li>
        </ol>

This is how it would look on a webpage:

  • First item
  • Second item
  1. First item
  2. Second item
