Lesson 1: Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create webpages. It structures content on the web by using tags that define elements like text, images, and links. In HTML, tags are used to mark the beginning and end of elements, and they help browsers display content correctly.

Example HTML Code:

        <html>
            <head></head>
            <body>
                <h1>Welcome to HTML!</h1>
                <p>This is an introduction to HTML.</p>
            </body>
        </html>

On the webpage, this code would display a heading saying "Welcome to HTML!" and a paragraph that says "This is an introduction to HTML." The <h1> tag defines the heading, and <p> defines a paragraph.
