Lesson 6: Tables and Images

The <img> tag will display an image. In the <img> tag, there are two parts, "src" and "alt". "Src" is the url or file path of the image while "alt" is a description of the image. The full tag will look like this <img src="image.jpg alt="Image"<>

Example of <img>tag:

<img src="image.jpg" alt="A beautiful image">

This is how it looks like on a webpage!

The <table> tag creates a table.In the <table> tag, there are the <tr>tag that defines a row, the <td> tag that defines the data inside the table, and the <th> tag that creates headers for the table.

Example of <table> tag:
 <table border="1">
            <tr>
                <th>Name</th>
                <th>Age</th>
                <th>City</th>
            </tr>
            <tr>
                <td>John</td>
                <td>25</td>
                <td>New York</td>
            </tr>
            <tr>
                <td>Jane</td>
                <td>30</td>
                <td>Los Angeles</td>
            </tr>
        </table>
This is how it looks like on a webpage!
